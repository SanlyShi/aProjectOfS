# my-app

## Project setup

```
安装node（自动安装npm）
安装cnpm
安装vue脚手架：npm install -g @vue/cli-service-global
使用脚手架创建项目：vue create hello-world (控制项目需要的配置选项)
```

```
cd my-app (打开项目文件夹)
npm install
```
